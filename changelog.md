# Gofas GalaxPay Pix

Módulo de gateway de pagamento para WHMCS que integra cobranças via Pix pela API GalaxPay. Desenvolvido pela Gofas Software.

## Funcionalidades

- Cobranças Pix via API GalaxPay
- QR Code dinâmico
- Baixa automática de fatura

## Requisitos

- WHMCS 7.x ou superior
- PHP 8.x
- Conta GalaxPay com Pix habilitado (Galaxy ID e Hash)

## Instalação

1. Copiar `modules/gateways/` para o `modules/gateways/` do WHMCS
2. Ativar e informar credenciais
